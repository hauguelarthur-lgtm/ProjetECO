GUIDE RAPIDE POUR LES AUTRES MEMBRES

1) Dossiers importants
- Cours/data/raw/ : fichiers sources (SIRENE, IRIS, Excel INSEE).
- Cours/data/processed/ : sorties generees par les scripts et notebooks.
- Cours/data/ : fichiers lourds OTP/transport (Graph, GTFS, OSM, jar).
- Cours/notebooks/ : analyses Rmd.
- Cours/scripts/ : scripts de preparation, comparaison et optimisation.

2) Fichiers minimaux requis pour lancer les notebooks principaux
- Cours/data/raw/georef-france-iris-millesime.{shp,shx,dbf,prj}
- Cours/data/raw/iris_isere.xlsx
- Cours/data/raw/dataseed-sirene-2.{shp,shx,dbf,prj}
  ou
- Cours/data/raw/dataseed-sirene-1.{shp,shx,dbf,prj,cpg}

3) Pour les notebooks OTP/isochrones (en plus)
- Cours/data/otp-1.5.0-shaded.jar
- Cours/data/Graph.obj (ou Cours/data/graphs/default/Graph.obj)
- Cours/data/gtfs/*
- Cours/data/rhone-alpes-latest.osm.pbf

4) Git
- Le repo ignore Cours/data/* sauf Cours/data/processed/** (voir .gitignore).
- Les gros fichiers de data restent en local (pas de push GitHub).
