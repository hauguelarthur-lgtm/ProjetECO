GUIDE RAPIDE POUR LES AUTRES MEMBRES

1) Dossiers importants
- Cours/data/raw/ : mettez ici les fichiers sources (shapefiles + excel).
- Cours/data/processed/ : sorties generees par les notebooks/scripts.
- Cours/data/ : fichiers OTP et gros fichiers locaux (graph, GTFS, OSM, jar).

2) Fichiers minimaux requis pour executer les notebooks principaux
- Cours/data/raw/dataseed-sirene-2.{shp,shx,dbf,prj}
- Cours/data/raw/georef-france-iris-millesime.{shp,shx,dbf,prj}
- Cours/data/raw/iris_isere.xlsx

3) Pour les notebooks OTP/isochrones (en plus)
- Cours/data/otp-1.5.0-shaded.jar
- Cours/data/Graph.obj (ou Cours/data/graphs/default/Graph.obj)
- Cours/data/gtfs/*
- Cours/data/rhone-alpes-latest.osm.pbf

4) Git
- Le repo est configure pour ignorer Cours/data/* sauf Cours/data/processed/**.
- Chacun garde ses gros fichiers de data en local sans les pousser sur GitHub.
